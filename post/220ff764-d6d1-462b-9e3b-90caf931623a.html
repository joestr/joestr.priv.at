2019 is over and 2020 is approaching

by Joel Strasser (joestr) on 2019-12-31T21:09Z

Hi,

The year 2019 was really intense for me. I contributed to various open source projects and gained a lot of experience. For example, I contributed to the Apache Archiva project, the Moquette project and to SemperVideos project called RPiList/specials. Some changes were only minor (especially the one for Moquette).

I also started a new project called PluGet. Basically said it is a Bukkit/BungeeCord plugin with the ability to install other plugins like with apt install <package name>.

Also my other projects like ZoneMenu (Spigot Plugin), DBWrapper and crtfind will receive updates.

Right now I'm in my final school year and I'm really busy at the moment but I will try to do as much as possible.

Enjoy the last moments of 2019 and see you in 2020 again!

Joel
