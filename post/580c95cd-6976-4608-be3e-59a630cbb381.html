Infrastructure domains explained

by Joel Strasser (joestr) on 2019-12-19T20:19Z

Hi,

Just for clarification: This blog entry explains the infrastrucuture of our servers. So let's begin:

There are two main domains involved in this process: gstd.eu and joestr.xyz

gstd.eu is a domain purely for the infrastrucutre. Servers for example get a name according to this schema:

[vm#.]host#.#.gstd.eu

The third virtual machine on the fourth server from the second hoster from Japan would get the following address:

vm3.host4.jpn2.gstd.eu

The infrastrucuture domain is also used for web panels and other infrastrucuture-like stuff.

joestr.xyz is a domain for the world to recognize.

You can see similarities with the attempt Alphabet/Google took:

1e100.net for the infrastrucuture and google.* for the world.

Thanks for reading!
